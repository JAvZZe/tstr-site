EXTRACTION METHODOLOGY
================================================================================

Search Strategies Used (in order of effectiveness):
1. Direct Certificate Search: site:customer.a2la.org "XXXX.XX" (~70% success)
2. Company Website Search: "Company Name" A2LA XXXX.XX (~60% success)
3. Organization + Certificate: A2LA "XXXX.XX" laboratory name (~40% success)
4. A2LA Directory Access: Direct portal links (~30% success)

Challenges Encountered:
1. PDF Encoding (15 PIDs): A2LA certificates stored as binary/compressed PDFs
   - Scope details visible but organization names not extractable
   - Example: Cert 0363.01 scope clear but lab identity hidden

2. Missing Certificate Numbers (8 PIDs): Source data gaps
   - No alternative identifier available for these PIDs
   - Requires manual A2LA directory verification

3. Scope Metadata Mismatch (8 PIDs): Certificate descriptions incorrect
   - Example: Cert 2778.01 labeled "Petroleum" but is EMC testing
   - Example: Cert 4795.01 labeled "Petroleum" but is auto components

4. International Labs (1 PID): ALSOUHUB (Kuwait) outside USA scope
